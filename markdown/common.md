##  keytool 工具的相关参数描述
``` bash
keypass/secret # 密钥口令
storepass/password # 密钥库口令
keytool -genkeypair -alias <alias> -keyalg RSA -dname "CN=Web Server,OU=Unit,O=Organization,L=City,S=State,C=US" -keypass <keypass> -keystore <output> -storepass <storepass>
```
> 有时 curl 命令请求成功，但数据结果没有打印在控制台上。可以通过把结果输出到文件来查看
``` bash
# curl 结果输出到 output.txt 文件中
curl http://xxxxxx/decrypt -d abcdefg > output.txt
```
