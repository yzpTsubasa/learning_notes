# double 类型的最大安全整数 9007199254740991
在计算机科学中，双精度浮点数（double类型）使用64位来表示一个数，其中1位用于符号位，11位用于指数，剩下的52位用于尾数（也称为有效数字或小数部分）。然而，由于浮点数的表示方法，尾数部分实际上隐含了一个前导的1，这样尾数部分实际上可以表示53位有效数字。

最大的安全整数是指在双精度浮点数中可以精确表示的最大整数，这个整数是2^53 - 1，即9007199254740991。这里的“安全”意味着在这个范围内的所有整数都可以被精确表示，而不会因为浮点数的精度限制而丢失信息。

具体来说，2^53 - 1这个值是53位，因为：

2^53是一个54位的数（从1开始计数），即1后面跟着53个0。
减去1后，得到的是53个1，这仍然是一个53位的数。
因此，尽管双精度浮点数的尾数部分只有52位，但由于隐含的前导1，实际上可以精确表示53位的整数。这就是为什么最大的安全整数是9007199254740991，它是53位的原因。
